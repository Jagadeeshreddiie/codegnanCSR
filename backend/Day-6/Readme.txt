Control Statements.

-->  Alter the flow of execution

**(if, switch, for, while, do-while)

if --> bi-directional branching Statement.
__________________________________________

Switch Case -->

    A multiway branching Statement
    By comparing a variable with multiple constraints and execute a specific branch matching the constraint.

    Break is not the part of Switch Case.

    If we don't use the break, then the flow will continue to the next blocks also.

_______________________________________________________________________________________________________________

While Loop ( entry controlled loop)  && Do-While Loop ( exit controlled loop) -->

Condition Controlled.

_______________________________________________________________________________________________________________

For Loop -->

finite loop having 

(Initialisation,Condition,Updation)

ini. -> test -> body -> update -> text etc..

_______________________________________________________________________________________________________________

Jumping Statements  -->

Break  &  continue

Break -> stops the flow of execution exactly.
Continue -> continues the flow by skipping the particular iteration.
